## Getting Started

1. **Clone the repository:**
   ```bash
   git clone https://github.com/sonikakannan/Dashboard-UI.git
   cd Dashboard-UI
   ```

2. **Install dependencies:**
   ```bash
   npm install
   # or
   yarn install
   ```

3. **Start the development server:**
   ```bash
   npm run dev
   # or
   yarn dev
   ```
